# AutoRent - Thuê Xe Tự Lái

AutoRent là một website giới thiệu dịch vụ thuê xe tự lái với giao diện tĩnh, có dữ liệu mẫu và nhiều trang nội dung để demo nhanh.

## Trang hiện có

- [xetulai.html](xetulai.html): Trang giới thiệu dịch vụ chính, danh sách xe mẫu và form đặt xe.
- [contact.html](contact.html): Trang liên hệ với thông tin mẫu và form gửi lời nhắn.
- [blogs.html](blogs.html): Trang blog chia sẻ mẹo thuê xe và tin tức mẫu.

## Nội dung mẫu

- Dữ liệu xe, giá thuê, quy trình đặt xe và đánh giá khách hàng được tạo sẵn để trang trông hoàn chỉnh hơn.
- Trang blog có sẵn danh sách bài viết mẫu, bài nổi bật và thống kê tổng quan.
- Trang liên hệ có hotline, email, địa chỉ và form phản hồi demo.

## Cách chạy

1. Mở trực tiếp file HTML bằng trình duyệt.
2. Hoặc dùng VS Code Live Server để xem các trang tĩnh.

## Ghi chú

- Dự án hiện đang dùng HTML, CSS và JavaScript thuần.
- Có thể nâng cấp sau bằng Flask hoặc lưu dữ liệu sang JSON/backend nếu cần.
